EPS = 稅後淨利(繳完稅之後的淨利) / 股數(資本額/10) ＝ 平均每一股完稅之後還賺多少錢 Earnings Per Share (每股稅後盈餘) 或是每股純益 每季為單位 簡單講 EPS 就是賺的錢 除以股票數量 本益比 = 股價 / EPS , 在EFT就是 所有成份股總市值 / 所有成份股總盈餘 （適用原型ETF） 股價淨值比 = 成份股的總市值 / 成份股的總淨值 （以1為基準 高於1 ）ETF的成份都是高成長型公司 股價淨值比自然比較高 00762元大未來 就是高成長型公司 BETA值 (看柴鼠 最佛系金控 )原則上 當BETA值 絕對值大於1越多 代表 相對於全市場波動越大 當BETA值 絕對值低於1越多 代表 波動相對低 如果是負值 代表和大盤相反 （季線60MA）BETA值用三個月當作一個基準 ROA = 稅後淨利(繳完稅之後的淨利) / 資產(經濟資源) Retrun On Asset 資產報酬率(整體資產的獲利能力) 簡單說 就是公司投入多少的資源產生這樣的獲利 ROE ＝ 稅後淨利(繳完稅之後的淨利) / 淨值(股東權益) Return On Equality 股東權益報酬率 因為公司的資產有部份可能是借貸來的錢 公司的資產 扣掉借來的錢 才是公司真正的錢 這就是 淨值 淨值在會計科目中叫做 股東權益 假設現在公司立刻 解散清算 資產和負債抵銷後 剩下的部份 就是每個股東 依照持股比例分配的權益
